# Whiteboard App

A collaborative whiteboard web application built with Next.js that includes real-time multi-user drawing capabilities and an AI chatbot assistant powered by Llama 3.3.

## Features

- **Drawing Area**: Interactive canvas with smooth drawing experience using Konva.js
- **Drawing Tools**: Pencil, brush, and eraser with customizable colors and stroke widths
- **Canvas Controls**: Undo, redo, clear canvas, with full history management
- **Save & Export**: Save drawings as JSON for later editing or export as PNG images
- **Real-time Collaboration**: Multiple users can draw simultaneously with live cursor tracking
- **AI Chatbot**: Integrated AI assistant using Groq's Llama 3.3 70B model (free tier)
- **Responsive Design**: Optimized layout for various screen sizes
